Triggering a form from a nav link

Pop a form modal from anywhere on the public site without writing code.

Open the Navigation admin and edit any top-level item or mega-menu link. Each link's edit panel includes a Triggers form dropdown that lists every form on this page. Pick one — the rendered link still carries its label / icon / button style, but clicking it opens the matching form modal instead of navigating to the URL. The link's URL stays as a no-JS / right-click fallback, so visitors with JavaScript disabled still reach the standalone form page.
