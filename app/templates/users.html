Roles & permissions

What each role can access. Module access is also gated per-module on the Modules tab.
  • Admin
    Full access. Manages users, modules, settings, the Web Frontend, and every other feature in the portal.
  • Editor
    Edits meetings, libraries, readings, and uploads. Cannot reach Settings, Users, or the Web Frontend module.
  • Frontend editor
    Inherits Editor permissions plus the entire Web Frontend module — header, footer, navigation, mega menus, alert bars, homepage content, theme/design tokens, and the public-visibility toggle.
  • Viewer
    Read-only. Sees meetings, libraries, files, and any module the admin opens to All signed-in users; cannot edit anything.
