What headers does it recognize?
  • Name — Name, Full Name, Display Name, or a pair of First Name + Last Name.
  • Email (required) — Email, Email Address, Mail.
  • Phone (optional) — Phone, Phone Number, Mobile, Cell, Telephone.

Rows missing a name or a usable email are skipped. Emails that already exist on the list are skipped — use the per-row Edit modal to update existing entries.
